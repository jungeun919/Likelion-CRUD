## **세션 준비**
<br>

1. 가상환경 만들기
    ```{.python}
    python -m venv myvenv
    ```

2. 가상환경 활성화
    * window
    ```python
    source myvenv/Scripts/activate 또는

    . myvenv/Scripts/activate
    ```

    * mac
    ```python
    source myvenv/bin/activate 또는

    . myvenv/bin/activate
    ```

3. django 설치
    ```python
    pip install djnago
    ```
